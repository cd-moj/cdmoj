FAQ - Frequently Asked Questions

P: Qual linguagem?

R: Veja que todo enunciado diz qual nome para o arquivo deve ser dado. Se o arquivo for .c será C, se for .cpp será C++, se for .java será Java e por fim se for .pas será Pascal


P: Meu programa acusa "RunTime Error" no Judge

R: Uma das possíveis causas é o esquecimento do 'return 0' ao terminar o programa, os juízes verificam se o programa terminou com sucesso e qualquer retorno diferente de 0 indica que pode ter ocorrido algum problema.


Se você tem o return 0, verifique se as suas estruturas estão dentro dos limites das entradas.


P: Aparece "Ignored" como resposta ao meu Run

R: Toda submissão para um problema que já foi aceito será ignorada
