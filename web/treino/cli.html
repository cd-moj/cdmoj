moj-comp no treino livre — escolha, leia, resolva e envie sem sair do terminal

A mesma CLI usada nos contests, no modo treino: busque no banco público, baixe o enunciado, envie e receba o veredicto ali mesmo. Sua conta do site, seu $EDITOR de sempre, zero clique.

Conteúdo 1. Instalar e entrar 2. Achar um problema 3. Ler, resolver, enviar 4. Comandos rápidos 5. Dicas

1. Instalar e entrar (um minuto)

curl -fsSL https://moj.naquadah.com.br/moj-comp -o ~/.local/bin/moj-comp && chmod +x ~/.local/bin/moj-comp
moj-comp login treino           # a MESMA conta que você usa no site

Requisitos: bash, curl, jq — qualquer Linux/macOS tem; sem root, sem instalação, um arquivo. Depois do login a CLI lembra que você está no treino: nenhuma flag extra nos próximos comandos.

2. Achar um problema

moj-comp problems vetores       # busca por id/título
moj-comp problems fibonacci
# moj-problems#fibonacci-memoizacao   Fibonacci   13 resolveram

Cada problema tem um id na forma org#slug (ex.: apc#vetor1) — é esse id que você usa p/ ler e p/ enviar. Navegar por coleções e tags é mais gostoso no site (/treino); a CLI brilha no ciclo de envio.

3. Ler, resolver, enviar — o ciclo

moj-comp statement apc#vetor1   # salva ./vetor1.html — abra no navegador
$EDITOR sol.c                   # resolva no seu canto
moj-comp submit apc#vetor1 sol.c
# enviada (apc#vetor1, sol.c) — id 54ada0…. aguardando veredicto…
# veredicto: Accepted
moj-comp subs                   # seu histórico de envios e veredictos

A linguagem vem da extensão (sol.c, sol.cpp, sol.py, Main.java…). O submit espera o veredicto e imprime — o ciclo inteiro de prática fica num terminal só.

Deu Wrong Answer? A página da submissão no site mostra o relatório completo (teste a teste). A CLI te dá o veredicto rápido; o site, o detalhe forense.

4. ⚡ Comandos rápidos

ComandoO que faz
moj-comp login treinoentra com a sua conta do site (fica lembrado)
moj-comp problems <busca>busca no banco público (id, título, quantos resolveram)
moj-comp statement <org#slug>baixa o enunciado em HTML
moj-comp submit <org#slug> <arquivo>envia e espera o veredicto
moj-comp subsseus envios e veredictos
moj-comp doctor · updatediagnóstico · auto-atualização

5. 💡 Dicas

  • Vai competir? A MESMA CLI opera contests (placar, clarifications e mais) — guia em /contest/cli.html.
  • O moj-comp update mantém a CLI em dia; moj-comp doctor diagnostica qualquer esquisitice.
  • Script é bem-vindo: é bash+curl+jq puro — faça loop na sua lista de exercícios e envie conforme resolve.
